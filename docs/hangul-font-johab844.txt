# 한글 11172자
- (초성 19자) * (중성 21자) * (종성 28자) = 11172자



# 초성/중성/종성
- 유니코드 한글 자소 분리
  초성: (n-0xAC00)/(21*28)
  중성: ((n-0xAC00)%(21*28))/28
  종성: (n-0xAC00)%28
