# 한글날 v2.0

임베디드 환경에서 한글 출력을 위한 라이브러리 입니다.  

한글 폰트 출력과 코드 변환 기능을 제공합니다.


## 특징

오직 순수 C 코드로만 작성되어 어떤 환경에서도 즉시 빌드됩니다.

DOS 시절 소프트웨어의 감성을 그대로 담은 조합형 한글 844 폰트 알고리즘을 사용하여, 

최소한의 데이터로 모든 한글을 표현합니다.



## 지원 기능

### 코드페이지 변환

한글 인코딩을 MCU 내부에서 즉시 처리합니다.

CP949: 한글 Windows 표준 호환

UCS-2 (UTF-16): 유니코드 16비트 평면 지원

UTF-8: 최신 웹/통신 표준 완벽 대응

### 문자 지원 범위

적은 메모리로 모든 한글을 표현합니다.

한글: 현대 한글 모든 마디 (11,172자) 및 낱글자 (51자)

한자: 상용 한자 (4,888자)

기호: 각종 특수 문자 (1,128자)
