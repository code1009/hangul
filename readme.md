# 훈민셈글

**셈글싸미 PC와 MCU가 달라,**  

**메모리가 서로 사맛디 아니할새.**  

이런 전차로 어린 개발자가 홀배 이셔도,  

**메모리가 넉넉지 못하야**  

마침내 제 뜻을 실어 한글 한 자 화면에  

올리지 못할 노미 하니라.  

내 이를 위하여 어엿삐 너겨, 

**오직 적은 메모리만 앗아가는 라이브러리를 맹가노니,**  

사람마다 하여 수비 니겨 날로 쑤매  

**밤샘 삽질**을 멈추게 하고저 할 따름이니라.





# 한글날 v2

한글날 v2은 아주 오래전, 밤을 지새우며 만들었던 한글 라이브러리에 대한 향수에서 시작되었습니다.

DOS 시절 우리에게 익숙했던 조합형 844 폰트 알고리즘을 그대로 가져오되, 

성능과 자원이 중요한 임베디드 환경에 맞춰 새롭게 설계했습니다. 

별도의 외부 라이브러리 없이 순수 C 코드만으로 구성되어 어디서든 가볍게 돌아갑니다.





## 지원 기능

### 코드페이지 변환

자체적으로 한글 인코딩을 내부에서 처리합니다.

- 한글 Windows(CP949) 표준 호환

- 유니코드 16비트 평면 UCS-2(UTF-16), UTF-8 지원



### 문자 지원 범위

적은 메모리로 모든 한글을 표현합니다.

- 한글: 현대 한글 모든 마디 (11,172자) 및 낱글자 (51자)

- 한자: 상용 한자 (4,888자)

- 기호: 각종 특수 문자 (1,128자)



### 한글 글자 입력

한글 입력 가능은 유니코드만 지원 합니다.
