# 훈민셈글

**셈글싸미 PC와 MCU가 달라,**  

**메모리가 서로 사맛디 아니할새.**  

이런 전차로 어린 개발자가 홀배 이셔도,  

**메모리가 넉넉지 못하야**  

마침내 제 뜻을 실어 한글 한 자 화면에  

올리지 못할 노미 하니라.  

내 이를 위하여 어엿삐 너겨, 

**오직 적은 메모리만 앗아가는 라이브러리를 맹가노니,**  

사람마다 하여 수비 니겨 날로 쑤매  

**밤샘 삽질**을 멈추게 하고저 할 따름이니라.





# 한글날 v2.0

임베디드 환경에서 한글 처리를 위한 **라이브러리**입니다.  

한글 폰트 출력과 코드 변환 및 입력 기능을 제공합니다.

아주 예전에 만들었던 한글 라이브러리를 추억을 되새기며,

임베디드 환경에 맞게 다시 만들어 보았습니다.



## 특징

오직 순수 C 코드로만 작성되어 어떤 환경에서도 즉시 빌드됩니다.

DOS 시절 소프트웨어의 감성을 그대로 담은 조합형 한글 844 폰트 알고리즘을 사용하여, 

최소한의 데이터로 모든 한글을 표현합니다.




## 지원 기능

### 코드페이지 변환

한글 인코딩을 MCU 내부에서 즉시 처리합니다.

- 한글 Windows(CP949) 표준 호환

- 유니코드 16비트 평면 UCS-2(UTF-16), UTF-8 지원



### 문자 지원 범위

적은 메모리로 모든 한글을 표현합니다.

- 한글: 현대 한글 모든 마디 (11,172자) 및 낱글자 (51자)

- 한자: 상용 한자 (4,888자)

- 기호: 각종 특수 문자 (1,128자)



### 한글 글자 입력

한글 입력 가능은 유니코드만 지원 합니다.
